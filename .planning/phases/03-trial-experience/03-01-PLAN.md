<objective>
Implement trial start at onboarding and for existing tenants, and lazy trial expiry so expired trials move to sandbox without blocking. Delivers TRIAL-01 and TRIAL-04.

Purpose: (1) create-tenant sets trial tier and trial_ends_at for new signups. (2) New RPC start_trial(p_tenant_id) allows existing sandbox tenants to start a trial. (3) get_tenant_licensed_modules at entry runs an UPDATE to set subscription_tier_id = 'sandbox' and trial_ends_at = NULL when subscription_tier_id = 'trial' and trial_ends_at < now(), then proceeds with normal module resolution. No new tables; no cron.
</objective>

<task type="auto">
  <name>Task 1: Migration — start_trial RPC and get_tenant_licensed_modules lazy expiry</name>
  <files>supabase/migrations/20260303000000_trial_start_rpc_and_expiry.sql</files>
  <action>
Create a single migration that:

1. **start_trial(p_tenant_id UUID)**  
   - LANGUAGE plpgsql SECURITY DEFINER SET search_path = public.  
   - If NOT (has_tenant_access(auth.uid(), p_tenant_id) OR has_role(auth.uid(), 'super_admin')) THEN RAISE EXCEPTION 'Access denied'.  
   - Optional: only allow if current row has subscription_tier_id = 'sandbox' (so trial is for sandbox tenants).  
   - UPDATE tenants SET subscription_tier_id = 'trial', trial_ends_at = now() + interval '30 days', updated_at = now() WHERE id = p_tenant_id AND subscription_tier_id = 'sandbox'.  
   - If no row updated, optionally raise or return (e.g. "Trial already active or not eligible").  
   - GRANT EXECUTE ON FUNCTION start_trial(uuid) TO authenticated.

2. **Extend get_tenant_licensed_modules**  
   - At the very start of the function body (after access check):  
     UPDATE public.tenants SET subscription_tier_id = 'sandbox', trial_ends_at = NULL, updated_at = now() WHERE id = p_tenant_id AND subscription_tier_id = 'trial' AND trial_ends_at IS NOT NULL AND trial_ends_at < now();  
   - Then continue with existing logic (RETURN SELECT ... tier_modules UNION override ...).

Use BEGIN/COMMIT in migration. Preserve existing get_tenant_licensed_modules signature and GRANT.
  </action>
  <verify>
    <automated>grep -q "start_trial\|trial_ends_at" supabase/migrations/20260303000000_trial_start_rpc_and_expiry.sql</automated>
  </verify>
  <done>start_trial RPC exists and is callable by tenant members; get_tenant_licensed_modules expires trial lazily.</done>
</task>
